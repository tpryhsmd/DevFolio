# 技術ポートフォリオアプリ

会社の技術情報を統一レイアウトでまとめ、静的HTMLとして書き出すElectronアプリ。

## 主要機能

- 写真・文章の配置（レイアウト固定）
- Undo/Redo
- バージョン履歴（複数保存・差分確認）
- 静的HTML書き出し
- 既存HTML読み込み・編集

## データ管理

独自拡張子によるシングルファイル管理。

## 開発

```bash
# 依存インストール（実装開始後）
npm install

# 起動
npm start
```

## ディレクトリ

```
.agent/    # AI運用メモ
.spec/     # 設計仕様
src/       # アプリ本体（実装開始時に作成）
app-dist/  # レンダラ成果物
release/   # 配布用成果物
```
